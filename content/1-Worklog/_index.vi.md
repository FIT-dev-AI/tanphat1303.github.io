# Nhật ký thực tập Cloud Engineer

## Tổng quan

Phần này ghi chép lại hành trình thực tập **15 tuần** tại **AWS Bootcamp - First Cloud AI Journey** với vị trí **Cloud Engineer intern**. Thực tập từ **17/04/2026** đến **30/07/2026**.

### Dự án: OCR-CapCut

Ứng dụng dịch thuật AI kết hợp OCR (Optical Character Recognition), được triển khai hoàn chỉnh trên hạ tầng AWS.

---

## Tiến độ hàng tuần

**Tuần 1:** [AWS Cloud Foundation & Làm quen](1.1-week1/)  
*Cơ bản về Cloud Computing, AWS Global Infrastructure, Thiết lập tài khoản*

**Tuần 2:** [SSH, Linux & Cơ bản EC2](1.2-week2/)  
*Quản lý Server, Command Line, Fundamentals EC2*

**Tuần 3:** [Phân tích dự án & Thiết kế kiến trúc AWS](1.3-week3/)  
*Lên kế hoạch OCR-CapCut, Docker Fundamentals, Sơ đồ kiến trúc*

**Tuần 4:** [Container hóa Docker & Triển khai Backend](1.4-week4/)  
*Tích hợp ECR, Docker Deployment lên EC2, CI/CD Pipeline*

**Tuần 5:** [Triển khai Frontend & CloudFront CDN](1.5-week5/)  
*S3 Static Hosting, CloudFront CDN, Cấu hình CORS*

**Tuần 6:** [VPC Networking & Bảo mật](1.6-week6/)  
*Mạng riêng tư, Application Load Balancer, Security Groups*

**Tuần 7:** [Tích hợp RDS & Secrets Manager](1.7-week7/)  
*Thiết lập Database, PostgreSQL Schema, Quản lý Credentials*

**Tuần 8:** [CI/CD Pipeline với GitHub Actions](1.8-week8/)  
*Tự động hóa hoàn toàn, GitHub Secrets, IAM Least Privilege*

**Tuần 9:** [CloudWatch Monitoring & Alerts](1.9-week9/)  
*Observability, Custom Metrics, SNS Notifications*

**Tuần 10:** [AWS AI Services - Textract & Translate](1.10-week10/)  
*Amazon Textract, Amazon Translate, Tích hợp AI/ML*

**Tuần 11:** [Tài liệu, Security & Tích hợp SES](1.11-week11/)  
*Viết tài liệu kỹ thuật, Security Hub, Amazon SES Email*

**Tuần 12:** [Load Testing, Auto Scaling & Báo cáo cuối kỳ](1.12-week12/)  
*Tối ưu hiệu suất, Auto Scaling, Hoàn thành thực tập*
