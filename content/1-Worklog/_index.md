# AWS Cloud Engineer Internship Worklog

## Overview

This section documents my 15-week internship journey at **AWS Bootcamp - First Cloud AI Journey** as a Cloud Engineer intern. The internship ran from **April 17, 2026** to **July 30, 2026**.

### Project: OCR-CapCut

An AI-powered translation application combining OCR (Optical Character Recognition) and machine translation, fully deployed on AWS infrastructure.

---
